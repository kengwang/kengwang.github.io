食用方法

主项目地址: https://github.com/nagadomi/waifu2x

有两个地方可以进行图片转换

网页版

蓝链直达: http://waifu2x.udp.jp/

(里面需要Google reCAPTCHA 可能在中国大陆无法使用)

页面也是十分的简单,注意放大倍数选择2x,点击转换之后就会跳转到新的一个地址

稍等片刻,一张新鲜的图片就出炉了!

速度亲测还是很快的,转换等待了10秒,下载速度视情况而定 (服务器在日本 可能对于某些环境会有点慢)

优点

  • 速度快 转换一会儿就好

  • 不耗费你自己的资源 (网络自费!)

  • 直接在他们的服务器上就转换好了,只需要下载了! (还需要动动手指哦)

  • 硬件依赖少 (网卡还是必须的呀!)

缺点

  • 可扩展性低

原版waifu2x有很多可以自选的参数,比如模式啊之类的,甚至可以选择不同的算法

  • 图片存储在服务器

这就涉及到了一点隐私问题了...... 假如我要转换一点那种的隐私图片,就不方便传到服务器上来下载.图片会被用来干什么谁也不知道(当然是训练模型啦! ?)

  • 追踪

看到下面的字了吗?

This website uses Google Analytics to help us improve the website content.

这就是Google的追踪了呀

  • reCAPTCHA

在进行转换的时候会验证是否是机器人,所有就会有Google reCAPTCHA的出现,某些东西我们知道的,所有我们就很难使用这个网站了

  • 白嫖的愧疚

白嫖使我快乐! 使用的时候会给他的服务器处理,而作者又要花大价钱续费&运营服务器

这就可以打赏一下作者了呀 Nagadomi的打赏页面

  • 无法批量转换

你只能一张一张的上传,一张一张地下载.不能上传多张图片一起下载

本地版

这个版本就要推荐一下他的第三方应用了

https://github.com/lltcggie/waifu2x-caffe

这个版本的参数很完备

最重要的是居然有中文,以至于不用Google翻译就可以知道内容了!

一般就直接选择输入路径,然后用他默认的输出路径

注意:

假如说你的显卡是NVIDIA系列的,请先安装CUDA驱动 (点我下载)

因为运算的话还是显卡快啊,但是也就只支持NVIDIA的显卡,A系列和I系列的就只有用CPU堆算力了!

假如你是NVIDIA的显卡,安装好驱动之后就点击waifu2x的应用设置... 处理器选择CUDA

其他的就选择CPU吧 ?

选择好了之后就可以点击转换了!转换的时候会有个进度条,下面也有个框框显示剩余时间.当然啦,NVIDIA的5s一张图,其他的就20多分钟一张图......

诶? 愿意为了花钱而提升速度?哈,真棒!

腾讯云GPU主机处理

这个亲测最少只用花1元多!

点击这个链接: 点我购买

点击了之后按理说不需要修改任何信息,直接点击下一步,在点击前请先确定价格是否正常 (1-9元每小时) 选择的机型是 GN6.7XLARGE48（GPU计算型GN6，28核48GB）

在购买之前,我们都知道按量计费,时间是金.我们需要做好以下准备:

  • VC++ Runtimes 建议用个什么一键包之类的,把全部都安装上

  • NVIDIA显卡驱动程序的下载地址

提供一下上面我的配置的选择:

Tesla -> P-Series -> Tesla P4 -> Windows Server 2016 -> 10.1 -> Chinese (Simplified)

不需要下载!直接在服务器上下载!

  • waifu2x主文件

因为在Github服务器的速度很慢,建议下载后再通过远程桌面传输到服务器

  • 7zip程序:

服务器与本地传输文件,松散的小文件速度很不可观,建议在互相传输之前都先压缩一下,7zip自身程序很小,也可以在服务器上直接下载,该程序强烈推荐!

  • Chrome安装程序

在线安装就可以了,因为服务器速度很快的!这个主要是用来下载驱动程序.

在准备好这些后就可以购买了,注意,你的余额一定要先充值购10元才可以购买,但是到时候扣款不会扣到10元的

购买后就可以用远程桌面mstsc连接了

连上了之后先检查网络,我的机子的DNS是手动指定的,所以需要调整到自动获取

然后将上述的文件上传到服务器,执行以下操作

  1. 安装VC++ Runtimes

  2. 安装Chrome

  3. 安装7-zip

  4. NVIDIA 驱动程序

  5. waifu2x主程序

开启的时候请注意下有没有什么报错之类的,提示什么DLL之类的多半是VC ++ 没有安装好,到百度直接搜索dll名称下载对应的VC++安装包

打开了之后就可以在应用设置里改成CUDA了,此时没有任何报错就代表成功,选择好文件改好设置就可以转换了!
